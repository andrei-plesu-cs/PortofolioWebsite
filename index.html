Projects Check some of my projects

Gymify

A mobile ready ideal companion for the gym, made with Angular and Bootstrap on the front end, and Firebase services on the back end. Being a Single Page Application, it also provides authentication and authorization, route guarding, validation, and quick access to the cloud database provided by Google through Firebase. The app is also a PWA as it's caching data through service workers for further offline ussage. Also, being PWA, it could be installed from the web to the home screen with just one click. Design of the app was made by the help of Adobe XD.

Technologies used

Html Sass Bootstrap Angular Typescript Firebase

HoolieChat

A chatting application made with the help of Angular and Bootstrap on the front end, while the back end is made with Node and Express. The core ideea is to use the Socket.io library both on the client and on the server side to allow an event-dirven communication between different clients that connect to the server. Also, the application is session-oriented so it does not provide persistent storage.

Technologies used

Html Css Bootstrap Angular Typescript Socket.io Node Express

myCompass

A replica of AirBnb made for the purpose of using APIs, such as Geolocation API for finding the exact position of the current user, Geocode API which translates coordinates to an address and most importantly, the Google Maps API, which maps locations based on their coordinates. Also, there is some vanilla javascript for some of the functionalities. Again, I've used Angular and Bootstrap on the front end, and Firebase on the back end.

Technologies used

Html Css Bootstrap Angular Typescript Firebase Google APIs
